Test Data Set
===============

.. meta::
   :keywords: test data


A test data set for |SourceXtractor++| covering various data reduction scenarios is available `here <https://cloud.physik.lmu.de/index.php/s/3K4KemBsw5y9yqd>`_. 
It contains the input data, all relevant configuration files, the |SourceXtractor++| commands and the expected output results for a reference.
